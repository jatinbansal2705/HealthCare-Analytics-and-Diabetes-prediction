# Healthcare Analytics and Diabetes Prediction

## Overview

This project demonstrates machine learning and exploratory data analysis techniques using diabetes healthcare datasets.

### Part 1 – Diabetes Prediction

- KNN-based missing value imputation
- Feature scaling
- Logistic Regression
- Accuracy, Precision, Recall
- Confusion Matrix
- ROC Curve

### Part 2 – Healthcare Data Analysis

- UCI Diabetes 130-US Hospitals Dataset
- HbA1c analysis
- Medication change analysis
- Readmission analysis
- Diagnosis grouping
- Heatmaps and bar charts

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset

- **Part 1:** `diabetes.csv` (included in this repository)
- **Part 2:** Downloaded automatically from the UCI Machine Learning Repository

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Open the notebook in Jupyter Notebook or Google Colab.

3. For Part 1, upload the provided `diabetes.csv` file when prompted.

4. Run all notebook cells.
